remix-etherscan-mcp

From verified source to local fork in one MCP surface.

This server turns Ethereum development and audit work into a single Claude-drivable tool plane. It combines Etherscan API v2, remixd workspace access, filesystem and git operations, compilation, JSON-RPC tracing, bytecode analysis, and Anvil-backed fork simulation.

Install
node 20+ · stdio first
local install

Build and run the server

The default transport is stdio, which is the normal path for Claude Desktop and Claude Code. 

git clone https://github.com/asuramaya/remix-etherscan-mcp.git
cd remix-etherscan-mcp
npm install
npm run build

export ETHERSCAN_API_KEY=your_api_key_here
node dist/index.js
http mode

Expose the MCP surface over HTTP

If you want streamable HTTP transport, pass --http and set MCP_PORT. 

MCP_PORT=3001 node dist/index.js --http
Prerequisites
optional capabilities stack
minimum

Core server requirements

  • Node.js 20 or later
  • Etherscan API key for explorer-backed tools
  • npx and local framework tools for compilation workflows
optional

Deeper audit capabilities

  • Foundry / Anvil for local fork simulation
  • Alchemy, Infura, or Geth-compatible RPC for trace and proof tools
  • Heimdall for bytecode decompilation
  • Slither and Vyper where those tool lanes matter
